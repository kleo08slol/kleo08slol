# Account switched to [Codeberg](https://codeberg.org/Kleo08s)
After the recent event that happened GitHub permanently suspended my [main GitHub account](https://github.com/Kleo08s) without any reason or warning.
That's why I decided to switch to a better alternative, [Codeberg](https://codeberg.org).

## Why switching?
You are may asking why not create a new GitHub account and continue the road here? Well, it's pretty simple why not.

Since GitHub is owned by Microsoft, it is bloated with AI slop, terrible support and with zero privacy protection. **That's why, every file you upload here (either private or not), Microsoft uses that data to train their AI model, GitHub Copilot.**

Codeberg instead doesn't have all that garbage since it is a non-profit and community-led project and they actually care about your privacy.

## So what about me?
I will be continue using GitHub but **only for big projects**, and if you want to see all of my projects you can check them out on [Codeberg](https://codeberg.org/Kleo08s?tab=repositories).
